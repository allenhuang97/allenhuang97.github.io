Pacman Game

This is my recreation of the arcade game Pacman.

It is an Applet that I created back in 2015 written in Java using Object Oriented Programming.

I used eclipse as my IDE along with photoshop for image editing.

Some Cool Features:

The game and title Page was fully animated

The game was accompanied by various audio tracks related to Pacman

The game contained a score screen recording highs scores from previous games. Stored locally in a .txt file

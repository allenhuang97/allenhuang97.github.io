WhatBroke Project

This project is a Web App that I developed with 3 of my friends for Hack the North 2016.

The Web App is a digitalized room condition form used by landlords to collect information from their tenants. We got the idea to make this after 3 out of 4 of us moved into a newly constructed apartment only to find several things were missing or damaged.

I worked on the front end of this web app using ReactJS and CSS. Overall this was a fun project to work on and was mainly a way for me to learn more about web development.

Above are some pictures of the front end of our web app. It consisted of a client side designed for tenants to enter their information and an admin side designed for landlords to view the data and generate reports.

Data Driven Prognostics Project

This project is what I worked on during my first co-op term as a Research Assistant.

All the code was written in R, Rcpp and C++.

Throughout my time working on this during my co-op I read several research papers and implemented many unique algorithms proposed in these papers. Later on I started to piece together my own methods for data driven prognostics.

In this project you will see several branches of machine learning and data conditioning. Some of these are: Hidden Markov Models, Loess Regression, Kalman Filters, Fast Fourier Transforms, SAX discretization method, etc.
